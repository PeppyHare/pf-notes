### Leap-Frog

n+1 --------|--------*--------|--------


n   --------*--------|--------*--------


n-1 --------|--------*--------|--------
            j-1      j        j+1

    Accuracy: O(Δt2,Δx2) O(\Delta t^2, \Delta x^2) O(Δt2,Δx2)

    Stability: It is marginally stable (G=1 G = 1 G=1) if ∣aΔtΔx∣≤1 \left| \frac{ a \Delta t}{\Delta x} \right| \leq 1 ∣∣∣​ΔxaΔt​∣∣∣​≤1

    It is also time-reversible, so we call it a symplectic integrator. This is a good thing! A time-reversible integrator will conserve energy.

    One big problem with Leap-Frog is the starting problem. How do we compute the very first iteration to get n=1 n=1 n=1 when we don't have information about n=−1 n=-1 n=−1? We need to "start" the algorithm by calculating the first step using a different method.

    Lax Algorithm

    ‘‘‘‘‘‘mathujn+1=uj+1n+uj−1n2−aΔtΔx(uj+1n−uj−1n2)``` ```math u_j ^{n+1} = \frac{u_{j+1} ^n + u_{j-1} ^n}{2} - \frac{a \Delta t}{\Delta x} \left( \frac{ u_{j+1} ^n - u_{j-1} ^n}{2} \right) ‘‘‘‘‘‘mathujn+1​=2uj+1n​+uj−1n​​−ΔxaΔt​(2uj+1n​−uj−1n​​)

    Accuracy: O(Δt,Δx2/Δt) O(\Delta t, \Delta x^2/\Delta t) O(Δt,Δx2/Δt)

    Stability: ∣aΔtΔx∣≤1 \left| \frac{ a \Delta t}{\Delta x} \right| \leq 1 ∣∣∣​ΔxaΔt​∣∣∣​≤1

    It may not be consistent if Δt→0 \Delta t \rightarrow 0 Δt→0 faster than Δx2→0 \Delta x^2 \rightarrow 0 Δx2→0

    We can re-write the Lax algorithm as

    ujn+1=ujn−aΔtΔx(uj+1n−uj−1n2)+uj+1n−2ujn+uj−1n2 u_j ^{n+1} = u_j ^n - \frac{ a \Delta t}{\Delta x} \left( \frac{ u_{j+1} ^n - u_{j-1} ^n}{2} \right) + \frac{ u_{j+1} ^n - 2 u_j ^n + u_{j-1} ^n}{2} ujn+1​=ujn​−ΔxaΔt​(2uj+1n​−uj−1n​​)+2uj+1n​−2ujn​+uj−1n​​

    Check out the last term - it's a difference operator for the second derivative. So what PDE do we approximate?

    ∂u∂t+a∂u∂x=Δx2Δt∂2u∂x2 \pdv{u}{t} + a \pdv{u}{x} = \frac{\Delta x^2}{\Delta t} \pdv{^2 u}{x^2} ∂t∂u​+a∂x∂u​=ΔtΔx2​∂x2∂2u​

    So we've added a second-derivative term to the linear advection equation. A second-derivative term acts as an artificial viscosity term, with viscosity Δx2/Δt \Delta x^2 / \Delta t Δx2/Δt.

    Lax-Wendroff Algorithm

    As we just saw, adding an artificial viscosity term can help to stabilize an unstable algorithm. If we take a Taylor series expansion

    ujn+1=ujn+Δtut+Δt22utt+Δt36uttt+… u_j ^{n+1} = u_j ^n + \Delta t u_t + \frac{ \Delta t ^2}{2} u_{tt} + \frac{\Delta t^3}{6} u_{ttt} + \ldots ujn+1​=ujn​+Δtut​+2Δt2​utt​+6Δt3​uttt​+…

    and use our governing equation to replace some of the time derivatives with spatial derivatives

    ut=−cux u_t = - c u_x ut​=−cux​

    utt=c2uxx u_{tt} = c^2 u_{xx} utt​=c2uxx​

    Then we get

    ujn+1=ujn−cΔtux+c2Δt22uxx−… u_{j} ^{n+1} = u_j ^n - c \Delta t u_x + \frac{c^2 \Delta t^2}{2} u_{xx} - \ldots ujn+1​=ujn​−cΔtux​+2c2Δt2​uxx​−…

    Substitute into expansion using centered finite difference operators

    ∂u∂x≈uj+1n−uj−1n2Δx∂2u∂x2≈uj+1n−2ujn+uj−1n2Δx2 \pdv{u}{x} \approx \frac{u_{j+1} ^n - u_{j-1} ^n}{2 \Delta x} \qquad \pdv{^2 u}{x^2} \approx \frac{u_{j+1} ^n - 2 u_j ^n + u_{j-1} ^n}{2 \Delta x^2} ∂x∂u​≈2Δxuj+1n​−uj−1n​​∂x2∂2u​≈2Δx2uj+1n​−2ujn​+uj−1n​​

    →ujn+1=ujn−aΔt2Δx(uj+1n−uj−1n)+a2Δt22Δx2(uj+1n−2ujn+uj−1n) \rightarrow u_j ^{n+1} = u_j ^n - \frac{ a \Delta t}{2 \Delta x} ( u_{j+1} ^n - u_{j-1} ^n) + \frac{ a^2 \Delta t^2}{2 \Delta x ^2}(u_{j+1} ^n - 2 u_j ^n + u_{j-1} ^n) →ujn+1​=ujn​−2ΔxaΔt​(uj+1n​−uj−1n​)+2Δx2a2Δt2​(uj+1n​−2ujn​+uj−1n​)

    Checking back up the page, we see the FTCS embedded directly in the algorithm, and we've got the same form of the second derivative but with a different multiplier. Instead of a 1 1 1 we've got a2Δt2Δx2 \frac{ a^2 \Delta t^2}{\Delta x^2} Δx2a2Δt2​

    What PDE are we approximating?

    ∂u∂t+a∂u∂x=a2Δt∂2u∂x2 \pdv{u}{t} + a \pdv{u}{x} = a^2 \Delta t \pdv{^2u}{x^2} ∂t∂u​+a∂x∂u​=a2Δt∂x2∂2u​

    Again we've added artificial viscosity, but now the viscosity only depends directly on Δt \Delta t Δt and goes to zero as Δt→0 \Delta t \rightarrow 0 Δt→0.

    We find that the Lax-Wendroff algorithm is O(Δt2,Δx2) O(\Delta t^2, \Delta x^2) O(Δt2,Δx2)

    n ------------------------*-------- j-1 j j+1

    
Another advantage of Lax-Wendroff over the Leap-Frog algorithm is that it avoids the starting problem. We don't need any information from step <span id='katexa553465fa7dd567c5c4bb3947dff9249'>katexa553465fa7dd567c5c4bb3947dff9249</span>, so it doesn't need to be started. How can that happen?

Write Lax-Wendroff as a 2-step method (predictor-corrector). Say we're going to use a Lax algorithm to advance to a midpoint


n+1    --------|-----------------*-----------------|--------


n+1/2  -----------------*-----------------*-----------------
                        j-1/2             j+1/2

n      --------|-----------------*-----------------|--------
               j-1               j                 j+1

    ‘‘‘‘‘‘‘‘‘mathuj+12n+12=uj+1n+ujn2−aΔtΔx(uj+1n−ujn2)``` ``` ```math u_{j+\frac{1}{2}} ^{n + \frac{1}{2}} = \frac{u^n _{j+1} + u_j ^n}{2} - \frac{a \Delta t}{\Delta x} \left( \frac{ u_{j+1} ^n - u_j ^n}{2} \right) ‘‘‘‘‘‘‘‘‘mathuj+21​n+21​​=2uj+1n​+ujn​​−ΔxaΔt​(2uj+1n​−ujn​​)

    ujn+1=ujn−aΔtΔx(uj+1/2n+1/2−uj−1/2n+1/2) u_j ^{n+1} = u_j ^n - \frac{a \Delta t}{\Delta x} \left( u_{j+1/2} ^{n+1/2} - u_{j-1/2} ^{n+1/2} \right) ujn+1​=ujn​−ΔxaΔt​(uj+1/2n+1/2​−uj−1/2n+1/2​)

    That's why Lax-Wendroff is the most popular explicit algorithm for linear PDEs.
