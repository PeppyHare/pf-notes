    Numerical Boundary Conditions

    The linear advection equation is

    ∂u∂t+a∂u∂x=0 \pdv{u}{t} + a \pdv{u}{x} = 0 ∂t∂u​+a∂x∂u​=0

    For it to be a well-posed mathematical problem, you also need the Cauchy data (boundary and initial conditions). From the order of the differential operators, we know that we need one boundary condition (constraint on u(xb) u(x_b) u(xb​) for all t t t ) and one initial condition (constraint on u(x) u(x) u(x) at t=0 t = 0 t=0).

    If we are solving the heat diffusion equation

    ∂T∂t=α∂2T∂x2 \pdv{T}{t} = \alpha \pdv{^2 T}{x^2} ∂t∂T​=α∂x2∂2T​

    Then we need 1 initial condition and 2 boundary conditions.

    Back to the linear advection equation, if we say

    u(t=0,x)=f(x)andu(t,x=0)=gD(t)(Dirichlet BC) u(t = 0, x) = f(x) \quad \text{and} \quad u(t, x=0) = g_D(t) \qquad \text{(Dirichlet BC)} u(t=0,x)=f(x)andu(t,x=0)=gD​(t)(Dirichlet BC)

    For wave speeds less than 0 (a<0 a < 0 a<0), then we would instead specify u(t,x=L) u(t, x=L) u(t,x=L).

    The way boundary conditions are implemented in the Forward Euler algorithm (as an example):

    ujn+1=ujn−aΔtΔx(ujn−uj−1n) u_j ^{n+1} = u_j ^n - \frac{ a \Delta t}{\Delta x} (u_j ^n - u_{j-1} ^n) ujn+1​=ujn​−ΔxaΔt​(ujn​−uj−1n​)

    1. We begin by specifying uj0=f(xj) u_j ^0 = f(x_j) uj0​=f(xj​), i.e. we populate the solution at t=0 t=0 t=0.
    2. Sweep through the domain starting at j=1 j=1 j=1 towards j=J j=J j=J

      u1n+1=u1n−aΔtΔx(u1n−u0n) u_1 ^{n+1} = u_1 ^n - \frac{a \Delta t}{\Delta x} (u_1 ^n - u_0 ^n) u1n+1​=u1n​−ΔxaΔt​(u1n​−u0n​)

      where

      u0n=gD(tn) u_0 ^n = g_D(t^n) u0n​=gD​(tn)

    3. Notice, at j=J j=J j=J

      uJn+1=uJn−aΔtΔx(uJn−uJ−1n) u_J ^{n+1} = u_J ^n - \frac{a \Delta t}{\Delta x} (u_J ^n - u_{J-1} ^n) uJn+1​=uJn​−ΔxaΔt​(uJn​−uJ−1n​)

      There is no boundary condition required since there is no data required from outside of our domain. If we do require such data, we'll need to do something special here.

    Leap-Frog BC

    Recall our Leap-frog algorithm

    ujn+1=ujn−1−aΔtΔx(uj+1n−uj−1n) u_j ^{n+1} = u_j ^{n-1} - \frac{a \Delta t}{\Delta x} (u_{j+1} ^n - u_{j-1} ^n) ujn+1​=ujn−1​−ΔxaΔt​(uj+1n​−uj−1n​)

    With assumed given initial condition

    uj0=f(x)=u(x,t=0) u_j ^0 = f(x) = u(x, t=0) uj0​=f(x)=u(x,t=0)

    What happens on the very first step from n=0 n=0 n=0 to n=1 n=1 n=1?

    uj1=uj−1−aΔtΔx(uj+10−uj−10) u_j ^1 = u_j ^{-1} - \frac{a \Delta t}{\Delta x} (u_{j+1} ^0 - u_{j-1} ^0) uj1​=uj−1​−ΔxaΔt​(uj+10​−uj−10​)

    We've got a starting problem: we need to know uj−1 u_j ^{-1} uj−1​, which is outside of our problem domain. In order to take the first time step, we bootstrap the algorithm by any some other algorithm which is not centered in time. Generally we use Lax-Wendroff to give us an additional initial condition uj1 u_j ^1 uj1​. With both uj0 u_j ^0 uj0​ and uj1 u_j ^1 uj1​, we can proceed with any second-order time-centered algorithm.

    How about the boundary conditions near the edges of the domain?

    u1n+1=u1n−1−aΔtΔx(u2n−u0n)(x=0) u_1 ^{n+1} = u_1 ^{n-1} - \frac{a \Delta t}{\Delta x} (u_2 ^n - u_0 ^n) \qquad (x = 0) u1n+1​=u1n−1​−ΔxaΔt​(u2n​−u0n​)(x=0)

    uJn+1=uJn−1−aΔtΔx(uJ+1n−uJ−1n)(x=L) u_J ^{n+1} = u_J ^{n-1} - \frac{a \Delta t}{\Delta x}(u_{J+1} ^n - u_{J-1} ^n) \qquad (x = L) uJn+1​=uJn−1​−ΔxaΔt​(uJ+1n​−uJ−1n​)(x=L)

    We're now being asked for information outside of the problem domain u0n u_0 ^n u0n​ and uJ+1n u_{J+1} ^n uJ+1n​. This goes back to the modified PDE that we're actually solving. The linear advection equation is first-order in x x x so we only need a single boundary condition to solve it, but the modified PDE that we are solving with Leap-frog has a second-order diffusive term, so we require an additional boundary condition.

    Our solution to the boundary problem is called a Numerical Boundary Scheme. As a zeroth-order scheme, we can just extend our solution at the boundary out one grid point

    (j=J)uJ+1n=uJ→uJn+1−aΔtΔx(uJn−uJ−1n) (j = J)\qquad u_{J+1} ^n = u_J \rightarrow u_J ^{n+1} - \frac{ a \Delta t}{\Delta x} (u_J ^n - u_{J-1} ^n) (j=J)uJ+1n​=uJ​→uJn+1​−ΔxaΔt​(uJn​−uJ−1n​)

    We could also make use of a first-order extrapolation

    uJ+1n=2uJn−uJ−1n→uJn+1=uJn−1−2aΔtΔx(uJn−uJ−1n) u_{J+1} ^n = 2 u_J ^n - u_{J-1} ^n \rightarrow u_J ^{n+1} = u_J ^{n-1} - \frac{2 a \Delta t}{\Delta x} (u_J ^n - u_{J-1} ^n) uJ+1n​=2uJn​−uJ−1n​→uJn+1​=uJn−1​−Δx2aΔt​(uJn​−uJ−1n​)

    Another method that's particularly well suited to leap-frog is

    uJ+1n=uJn−1 u_{J+1} ^n = u_J ^{n-1} uJ+1n​=uJn−1​

    The advantage of this method is that in a centered scheme it preserves information flow. It assumes that information from (n−1,J) (n-1, J) (n−1,J) will flow to (n,J+1) (n, J+1) (n,J+1), and we can use that information to compute (n+1,J) (n+1, J) (n+1,J). This helps to prevent unphysical reflections at the boundary.

    Another concept which addresses unphysical boundary effects that we will cover later is called PML (Perfectly Matched Layer).

    NBS are needed in all algorithms which use central difference operators.

    Neumann Boundary Conditions

    Instead of specifying the value of u u u at the boundary, we can specify the value of ∂u∂x \pdv{u}{x} ∂x∂u​ instead

    ∂u∂x∣x=0=gN(t) \left.\pdv{u}{x}\right|_{x=0} = g_N(t) ∂x∂u​∣∣∣∣∣​x=0​=gN​(t)

    →u1n−u0nΔx=gN(t)→u0n=u1n−ΔxgN(t) \rightarrow \frac{u_1 ^n - u_0 ^n}{\Delta x} = g_N(t) \rightarrow u_0 ^n = u_1 ^n - \Delta x g_N(t) →Δxu1n​−u0n​​=gN​(t)→u0n​=u1n​−ΔxgN​(t)

    What happens when we put this into the Forward Euler algorithm

    u1n+1=u1n−aΔtΔx(u1n−[u1n−ΔxgN(t)])=u1n−aΔtgN(t) u_1 ^{n+1} = u_1 ^n - \frac{a \Delta t}{\Delta x} \left(u_1 ^n - \left[u_1 ^n - \Delta x g_N(t)\right]\right) = u_1 ^n - a \Delta t g_N (t) u1n+1​=u1n​−ΔxaΔt​(u1n​−[u1n​−ΔxgN​(t)])=u1n​−aΔtgN​(t)

    So the spatial difference has been entirely replaced by the boundary condition, which is about what we would expect. It means that we can essentially remove j=1 j=1 j=1 from our algorithm, since we can write down the value of u1n u_1 ^n u1n​ for all n n n only using the BC and IC.

    Consider j=1 j=1 j=1, and define ν≡aΔt2Δx \nu \equiv \frac{a \Delta t}{2 \Delta x} ν≡2ΔxaΔt​ to clean things up

    (j=1)u1n+1+θνu2n+1=u1n−(1−θ)νu2n+ν[θu0n+1+(1−θ)u0n] (j = 1) \quad u_1 ^{n+1} + \theta \nu u_2 ^{n+1} = u_1 ^n - (1 - \theta) \nu u_2 ^n + \nu [\theta u_0 ^{n+1} + (1 - \theta) u_0 ^n] (j=1)u1n+1​+θνu2n+1​=u1n​−(1−θ)νu2n​+ν[θu0n+1​+(1−θ)u0n​]
