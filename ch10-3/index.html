    Lagrange Multipliers

    When we have a problem of variation where one function is maximized or minimized subject to a constraint imposed by another function

    f(x1,…xn)→ function  f(x_1, \ldots x_n) \rightarrow \text{ function } f(x1​,…xn​)→ function 

    g(x1,…,xn)=0→ constraint  g(x_1, \ldots, x_n) = 0 \rightarrow \text { constraint } g(x1​,…,xn​)=0→ constraint 

    Without the constraint we would have the problem

    δf=∂f∂x1δx1+…∂f∂xnδxn \delta f = \pdv{f}{x_1} \delta x_1 + \ldots \pdv{f}{x_n} \delta x_n δf=∂x1​∂f​δx1​+…∂xn​∂f​δxn​

    ∂f∂xi=0i=1…n \pdv{f}{x_i} = 0 \qquad i = 1 \ldots n ∂xi​∂f​=0i=1…n

    and apply any of our multivariate optimization strategies to solve. This can be hard. Luckily, at any stationary point of the function that also satisfies the constraint, the gradient of the function at that point can be expressed as a linear combination of the gradients of the constraints at that point.

    ∂f∂xi−λ∂g∂xi=0 \pdv{f}{x_i} - \lambda \pdv{g}{x_i} = 0 ∂xi​∂f​−λ∂xi​∂g​=0

    With a constraint g=0 g = 0 g=0, the change in f f f becomes a change in the functional of f f f and g g g.

    f,g→h(f(x1,…,xn−1),g(x1,…,xn−1))=h(x1,…,xn−1)=0 f, g \rightarrow h(f(x_1, \ldots, x_{n-1}), g(x_1, \ldots, x_{n-1})) = h(x_1, \ldots, x_{n-1}) = 0 f,g→h(f(x1​,…,xn−1​),g(x1​,…,xn−1​))=h(x1​,…,xn−1​)=0

    ∂h∂xi→i=1→n−1 \pdv{h}{x_i} \rightarrow i = 1 \rightarrow n-1 ∂xi​∂h​→i=1→n−1

    f=A=x1x2 f = A = x_1 x_2 f=A=x1​x2​

    g=0=2x1+2x2−4a g = 0 = 2x_1 + 2x_2 - 4a g=0=2x1​+2x2​−4a

    δf=x2δx1+x1δx2 \delta f = x_2 \delta x_1 + x_1 \delta x_2 δf=x2​δx1​+x1​δx2​

    λδg=λ2δx1+λ2δx2 \lambda \delta g = \lambda 2 \delta x_1 + \lambda 2 \delta x_2 λδg=λ2δx1​+λ2δx2​

    x2+2λ=0x1+2λ=0 x_2 + 2\lambda = 0 \qquad x_1 + 2\lambda = 0 x2​+2λ=0x1​+2λ=0
